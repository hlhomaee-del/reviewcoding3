📚 Key Infectious Disease Terms — Click to Expand
Infection▶
(in-FEK-shun)
Invasion and multiplication of microorganisms in the body causing disease. For coding: identify the TYPE of organism and SITE affected — both often drive the code. Main term = usually the infection type or disease name (e.g., Cellulitis, Pneumonia, Sepsis)
Sepsis▶
(SEP-sis)
Life-threatening condition where the body's response to infection becomes dysregulated. For coding: identify severity (sepsis vs. severe sepsis vs. septic shock) and the causative organism. Sequencing is critical. Main term: Sepsis → organism subterm (e.g., due to Staphylococcus)
Cellulitis▶
(sel-yoo-LY-tis)
Bacterial skin infection causing redness, swelling, warmth in deeper skin layers. For coding: the SITE and LATERALITY are the primary code drivers. Right vs left, lower leg vs foot — all matter. Main term: Cellulitis → subterm: body site → laterality
Septicemia vs. Sepsis▶
(sep-tih-SEE-mee-uh)
"Septicemia" is an OLDER term that ICD-10-CM does NOT use as a main entry. When a provider writes "septicemia," the coder must look up "Sepsis" in the index. This is a key example of provider wording NOT matching ICD-10-CM index terminology. Key rule: "Septicemia" → look up Sepsis in the Index
Abscess▶
(AB-ses)
Localized collection of pus from bacterial infection. For coding: the SITE of the abscess drives the code. Skin abscess, peritonsillar abscess, liver abscess = all different codes. Main term: Abscess → subterm: site (skin, finger, peritonsillar, liver, etc.)
Urinary Tract Infection (UTI)▶
(yur-ih-NAIR-ee trakt)
Infection of any urinary system part. For coding: identify WHERE in the urinary tract (cystitis = bladder, pyelonephritis = kidney, NOS = unspecified). Organism, if documented, further specifies the code. Main term: Cystitis, Pyelonephritis, or Urinary tract infection

A provider documents "septicemia due to Staphylococcus aureus." When looking this up in the Alphabetic Index, what main term should the coder use?
ASepticemia — use the provider's exact word
BSepsis — "septicemia" maps to Sepsis in ICD-10-CM; look up Sepsis first
CStaphylococcus — the organism is always the main term
DBacteremia — it means the same thing as septicemia
💡 Rationale
"Septicemia" is an older clinical term that ICD-10-CM does NOT list as its primary index entry. When a provider writes "septicemia," the coder looks up "Sepsis." This is a critical example of provider documentation wording NOT matching the ICD-10-CM index entry. The coder must know to translate "septicemia" → "Sepsis." The subterm "due to Staphylococcus aureus" then further specifies the code (A41.01).
🧫 Foundations — Q2
Why might a provider's documentation wording NOT match the exact wording in the ICD-10-CM Alphabetic Index?
AProviders are not trained in ICD-10-CM terminology
BProviders use clinical/medical terminology that may be synonyms, older terms, or highly specific — while ICD-10-CM may organize the same condition under a different term
CThe ICD-10-CM Index only uses Latin terms
DCoders do not need to know medical terminology to code
💡 Rationale
Providers use clinical language reflecting their training — synonyms, older terms, or highly specific medical terminology. ICD-10-CM may list the same condition under a different term (e.g., "septicemia" → look up "Sepsis"; "blood poisoning" → look up "Sepsis"; "flu" → look up "Influenza"). Coders must know medical terminology AND how ICD-10-CM organizes conditions to bridge this gap. Medical terminology knowledge is essential for accurate coding.

🔬 Why Organism Type Affects Code Selection
🦠 Bacteria
Specific bacteria → specific codes. "Bacterial pneumonia" (J15.9) vs. "Staphylococcal pneumonia" (J15.20) vs. "Pneumococcal pneumonia" (J13). The organism name drives the subterm path.
🔬 Virus
Viral infections have dedicated code categories. Influenza (J09-J11), COVID-19 (U07.1), RSV (J06.9), HIV (B20). The specific virus drives which category is used.
🪱 Parasite
Chapter 1 covers many. Malaria (B50-B54), Toxoplasmosis (B58), Intestinal parasites (B76-B83). Specific parasite AND site both matter.
🍄 Fungus
Fungal infections (mycoses) B35-B49. Candidiasis by site (B37.-), Aspergillosis (B44.-), skin fungi (B35.-). Both organism AND site drive the code.

🦠 Organism — Q1
A provider documents "candidal vaginitis." Candida is a fungus. What is the best coding approach?
ALook up Vaginitis → bacterial → Candida
BLook up Candidiasis → vulvovaginitis (B37.3) OR Vaginitis → candidal → B37.3; knowing Candida is a fungus guides the correct subterm path
CLook up Parasite → Candida → reproductive system
DLook up Fungal infection NOS → female
💡 Rationale
Knowing Candida is a fungus helps the coder navigate correctly — not looking under "bacteria" or "parasite." Two valid index paths both lead to B37.3: (1) Candidiasis → vulva and vagina; or (2) Vaginitis → candidal. Either entry works. Knowing organism types prevents the coder from looking in the wrong section of the index. Always verify in the Tabular List after finding the index code suggestion.
🦠 Organism — Q2
A provider documents "UTI due to E. coli." How does documenting the specific organism affect coding compared to just "UTI"?
AIt doesn't change anything — UTI codes are the same regardless of organism
BIt allows a more specific code AND may require an additional organism code (B96.20 for E. coli) per "Use additional code" instructions in the Tabular
CE. coli is only coded if the patient is hospitalized
DThe organism replaces the UTI code entirely
💡 Rationale
Documenting E. coli adds specificity AND triggers an additional code requirement. The Tabular for UTI/cystitis codes often has a "Use additional code" instruction to identify the organism using B95-B97 codes. E. coli = B96.20. The result: one code for the infection site/type (e.g., N30.00 for acute cystitis) PLUS B96.20 for the organism. Specific documentation = more complete, accurate coding.

Key Concept: Sometimes the SITE is the primary code driver (cellulitis, abscess, UTI site). Other times, the DISEASE NAME or ORGANISM is the primary driver (tuberculosis, influenza, HIV). Analyze the documentation to determine which applies.

🫁 Body System — Q1
A provider documents "cellulitis of the right foot." What is the PRIMARY driver of code specificity?
AThe causative organism — which bacteria caused the cellulitis
BThe site and laterality — "right foot" drives which cellulitis code is selected
CThe patient's age — pediatric and adult cellulitis use different codes
DThe severity — mild vs severe cellulitis changes the code
💡 Rationale
For cellulitis, SITE and LATERALITY are the primary code drivers. Index path: Cellulitis → foot → right foot → L03.011. Left foot = L03.012. The organism, while clinically important, is not always separately coded for cellulitis unless documented and unless the Tabular has a "Use additional code" instruction. Laterality (right vs left) is critical and must match the documentation exactly.
🫁 Body System — Q2
A provider documents "bacterial meningitis due to Streptococcus pneumoniae." What drives the MOST SPECIFIC code?
AThe site alone — meningitis always uses the same code regardless of cause
BBoth site AND organism — bacterial meningitis codes are specific to the causative organism (Streptococcal meningitis = G00.1)
CThe term "bacterial" only — no further specificity needed
DCode the symptoms — headache and fever drive the code
💡 Rationale
For meningitis, BOTH the site (meninges) AND the organism determine the code. "Bacterial meningitis NOS" = G00.9 (unspecified). "Streptococcal meningitis" = G00.1 (more specific). Unlike cellulitis where the site alone is usually sufficient, in meningitis the organism type (bacterial, viral, fungal) AND the specific organism both shape code selection. Specific documentation → specific code → better clinical capture.
🫁 Body System — Q3
A patient is diagnosed with "pulmonary tuberculosis." The disease name already implies the organism (Mycobacterium tuberculosis). What is the correct coding approach?
ALook up "Mycobacterium" first — organism is always the main term
BLook up "Tuberculosis" (the disease name) as the main term, with "pulmonary" as the subterm — the disease name encompasses the organism
CLook up "Pulmonary" first — site drives the main term
DCode the symptoms — cough and hemoptysis drive TB coding
💡 Rationale
Some disease names inherently include the organism. "Tuberculosis" = caused by Mycobacterium tuberculosis — this is embedded in the disease name. Index path: Tuberculosis → pulmonary → A15.0. The coder does NOT look up "Mycobacterium" separately. The disease name IS the main term. The subterm "pulmonary" narrows the site. This is the case for many named diseases (Lyme disease, Salmonellosis, Shigellosis, etc.).

🔍 Main Term Decision Guide — 3 Questions
Ask yourself before looking up any code:
1️⃣ Is there a specific named disease that encompasses everything? (Tuberculosis, Lyme disease, HIV, Influenza, C. diff) → Use the disease name as main term

3️⃣ Is there a known organism + non-specific infection term? (Staphylococcal infection, Klebsiella infection) → Try BOTH the organism AND the infection type to find the best path

🔍 Main Term — Q1
A provider documents "infection due to Clostridium difficile (C. diff) with pseudomembranous colitis." What is the correct main term in the ICD-10-CM Alphabetic Index?
AColitis — the colon is the primary body site
BInfection — the general infection category
CEnterocolitis — ICD-10-CM classifies C. diff under Enterocolitis → due to Clostridium difficile → A04.7_
DClostridium — the organism name is always first
💡 Rationale
C. difficile colitis is classified under "Enterocolitis due to Clostridium difficile" (A04.7_) in ICD-10-CM. The main term is "Enterocolitis" — not "colitis," not "Clostridium," not "infection." This is a prime example of provider terminology ("pseudomembranous colitis") not matching the ICD-10-CM index term ("Enterocolitis"). The coder must know both the clinical term AND how ICD-10-CM organizes it to find the correct code.
🔍 Main Term — Q2
A provider documents "MRSA infection of a surgical wound." What should the coder know about how to code this?
AMRSA is coded with a single code covering the infection and organism together
BMRSA is treated identically to any other Staphylococcal infection — same code
CThe wound infection code is assigned for the site, PLUS an additional code for MRSA as the causative organism (B95.62) — the Tabular typically requires both
DMRSA is never separately coded — it is clinical information only
💡 Rationale
MRSA coding requires specific attention. Many infection codes in the Tabular have a "Use additional code" instruction to identify MRSA when it is the causative organism — typically B95.62 (MRSA as the cause of diseases classified elsewhere). The wound infection code is based on the site/type, and MRSA is separately identified. This is why reading ALL Tabular notes matters — you would miss the MRSA additional code requirement if you stopped at the index.

⚡ Code Driver — Q1
A provider documents "fever and cough" at the start of the note, then the Assessment states: "Influenza A, confirmed." What should be coded?
AFever and cough — they were documented first
BInfluenza A — the confirmed Assessment diagnosis; fever and cough are integral symptoms not separately coded when the confirmed diagnosis is documented
CBoth the symptoms AND Influenza A — all documented findings must be coded
DInfluenza A is only coded if confirmed by a rapid flu test result
💡 Rationale
The confirmed Assessment diagnosis (Influenza A) drives code selection for the entire encounter — even though the symptoms were documented earlier in the note. Fever and cough are classic integral symptoms of influenza — they are NOT separately coded when the confirmed diagnosis is documented. The entire note is read, and the Assessment/confirmed diagnosis always takes priority over symptom coding when both are present in the same note.
⚡ Code Driver — Q2
A note's Assessment says "skin abscess, left arm." Later in the SAME note, culture results are documented: "Staphylococcus aureus." Can the coder use the organism information from later in the note?
ANo — only information in the Assessment section can be coded
BYes — all documentation within the same encounter note can inform code selection; the Tabular may require an additional organism code when the organism is documented anywhere in the note
CWait for a new visit note to code the organism
DCode only the organism — ignore the "skin abscess" entry
💡 Rationale
All documentation within the SAME encounter note can be used by the coder — not just the Assessment. If culture results appear later in the note, the coder can use this information. The Tabular for skin abscess codes often has a "Use additional code" instruction for the causative organism — allowing the coder to add B95.61 (Staphylococcus aureus as the cause). The causative organism can be documented ANYWHERE in the same encounter note and is fair to code.
⚡ Code Driver — Q3
Which documentation phrase would produce the MOST SPECIFIC and COMPLETE set of ICD-10-CM codes for a sepsis case?
A"Sepsis" — general diagnosis
B"Sepsis due to bacteria" — identifies organism type only
C"Sepsis due to Staphylococcus aureus" — specific organism identified
D"Severe sepsis due to Staphylococcus aureus with septic shock" — identifies organism, severity, AND complication → multiple specific codes required
💡 Rationale
"Severe sepsis due to Staphylococcus aureus with septic shock" provides the most code drivers: (1) Staphylococcal sepsis code (A41.01), (2) Severe sepsis (R65.20 or R65.21), (3) Septic shock (R65.21 or T81.12XA), potentially (4) B95.61/B95.62 for MRSA if applicable. Each documented detail — organism, severity, presence of shock — adds codes and clinical accuracy. Comprehensive provider documentation directly enables comprehensive, accurate coding.

ANo — E. coli is conditional ("if confirmed") and not yet confirmed in this note; code only confirmed diagnoses

📄 Note 3 — Q1
How many separate code categories are needed for Ms. Abara's Assessment?
AOne — sepsis covers all of her conditions
BTwo — sepsis and pyelonephritis
CAt least three to four: sepsis code (organism NOS for now), pyelonephritis as the source, severe sepsis code, septic shock code — plus diabetes as a comorbidity

Each documented element that affects the patient's care is coded. Waiting for culture results before coding anything is incorrect — code what is CONFIRMED NOW with the most specific codes available.
📄 Note 3 — Q2
The Assessment states "causative organism unknown at this time." Can the coder still assign a sepsis code?
AYes — A41.9 (Sepsis, unspecified organism) is assigned when the organism is not identified; the confirmed diagnosis is sepsis regardless of organism identification

COnly code sepsis if the patient is discharged with the final diagnosis
DCode only the pyelonephritis until the organism is confirmed
💡 Rationale
ICD-10-CM provides A41.9 (Sepsis, unspecified organism) for exactly this situation — when the provider has confirmed SEPSIS but the causative organism is not yet identified. When culture results come back and confirm the organism, the code can be updated to the organism-specific code. Code the confirmed diagnosis NOW with the best available specificity.

COVID-19 with pneumonia requires multiple codes. Per ICD-10-CM instructions: U07.1 (COVID-19) and J12.82 (Pneumonia due to COVID-19). This is the etiology/manifestation convention in action — the underlying disease (COVID-19) is coded first, followed by the manifestation (pneumonia). The Tabular for COVID-19 has specific sequencing instructions. Always verify in the Tabular for current COVID-19 coding guidelines as these have evolved.
